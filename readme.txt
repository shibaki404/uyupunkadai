=======================================
  郵便番号検索アプリ - 使い方と説明
=======================================

【概要】
郵便番号を入力すると、自動的に対応する住所を表示するWebアプリケーションです。
無料の郵便番号APIを使用して、リアルタイムで住所を検索します。

【ファイル構成】
├── index.html    メインのHTMLファイル
├── script.js     JavaScriptファイル（郵便番号検索機能）
├── style.css     スタイルシート（デザイン）
└── readme.txt    このファイル

【必要な環境】
- Webブラウザ（Chrome, Firefox, Edge, Safari など）
- インターネット接続（郵便番号APIへのアクセスに必要）

【使い方】
1. index.html をWebブラウザで開く
   - ダブルクリックするか、ブラウザにドラッグ＆ドロップ

2. 郵便番号を入力
   - 7桁の数字をハイフンなしで入力
   - 例: 1000001（東京都千代田区千代田）
   - 例: 5300001（大阪府大阪市北区梅田）
   - 例: 0600000（北海道札幌市中央区）

3. 検索ボタンをクリック
   - または、Enterキーを押す

4. 住所が自動的に表示される
   - 都道府県
   - 市区町村
   - 町域
   - 住所（全体）

【主な機能】
✓ 郵便番号から住所を自動検索
✓ 数字以外の入力を自動除去
✓ Enterキーで検索可能
✓ エラーメッセージ表示
✓ レスポンシブデザイン（スマホ対応）

【使用している技術】
- HTML5
- CSS3
- JavaScript（ES6+）
- zipcloud API（https://zipcloud.ibsnet.co.jp/）

【エラーメッセージについて】
「郵便番号は7桁で入力してください」
  → 6桁以下または8桁以上の数字が入力されています

「該当する住所が見つかりませんでした」
  → 入力された郵便番号が存在しません

「住所の取得に失敗しました。ネットワーク接続を確認してください。」
  → インターネット接続に問題があります

【郵便番号の例】
1000001 - 東京都千代田区千代田
1000014 - 東京都千代田区永田町
1500002 - 東京都渋谷区渋谷
5300001 - 大阪府大阪市北区梅田
6008216 - 京都府京都市下京区東塩小路町
2310023 - 神奈川県横浜市中区山下町
8100001 - 福岡県福岡市中央区天神
0600000 - 北海道札幌市中央区

【注意事項】
- インターネット接続が必要です
- 郵便番号データは外部APIから取得しています
- APIの利用制限により、短時間に大量の検索はできません
- ブラウザのセキュリティ設定により、ローカルファイルから
  APIにアクセスできない場合は、Webサーバー経由で開いてください

【ローカルサーバーでの起動方法】
もしブラウザで直接開いてAPIエラーが出る場合は、
以下のいずれかの方法でローカルサーバーを起動してください：

方法1: Python 3を使用
  コマンドプロンプトでこのフォルダに移動し、
  python -m http.server 8000
  その後、ブラウザで http://localhost:8000 を開く

方法2: Node.jsのhttp-serverを使用
  npm install -g http-server
  http-server
  その後、表示されるURLをブラウザで開く

方法3: Visual Studio Code のLive Server拡張機能を使用
  拡張機能をインストールし、index.htmlを右クリックして
  「Open with Live Server」を選択

【カスタマイズ】
- style.css を編集することで、デザインを変更できます
- script.js を編集することで、機能を追加・変更できます
- 色やレイアウトは自由にカスタマイズ可能です

このアプリケーションは自由に使用・改変できます。
